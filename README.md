## About / О себе

**EN** — By day: Technical Specialist at Prima Power, maintaining and training on CNC sheet-metal equipment across KSA. By night: Python developer who builds Telegram bots, backend services and IoT tools that solve real production problems.

I write code the same way I approach a machine fault — diagnose to the root, fix it properly, document it. Most of my projects come from actual operational needs: production downtime tracking, VPN access management, real-time furnace monitoring. Not tutorial clones.

I use AI tooling (Gemini, Claude, Antigravity) as a standard part of the workflow — for code generation, refactoring, architecture reviews and automating repetitive tasks. It speeds up experiments and raises quality; all architectural decisions stay with me.

Не "AI написал за меня" — а AI как постоянный инструмент в конкретных задачах:

- **Claude Code + Antigravity Gemini** — рефакторинг и code review, генерация boilerplate, автоматизация повторяющихся задач. Antigravity используется как агент с долгосрочным контекстом на крупных проектах
