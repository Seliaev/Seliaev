## Denis Seliaev

В свободное время — разрабатываю ботов, веб-инструменты и автоматизацию для бизнеса.

Большинство проектов выросли из реальных задач: учёт простоев станков, мониторинг печей, выдача VPN-ключей.  
Остальные — фриланс: лендинги и боты под ключ за 2–5 дней.
